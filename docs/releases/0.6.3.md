# spectral-library 0.6.3

## Highlights

- built-in Sentinel-2A, Sentinel-2B, and Sentinel-2C canonical `nir` now map
  to `B08` from `rsrf>=0.3.1`, matching the SIAC-facing semantic band contract
- the prepared-runtime schema advanced to `3.1.0` so previously published
  `0.6.2` runtimes using Sentinel `nir -> B8A` are rejected instead of being
  reused silently
- the official Sentinel-2A example bundle, docs, and packaged prepared runtime
  were regenerated against the new `B08` mapping

## Upgrade Notes

- rebuild and republish any prepared runtime that includes `sentinel-2a_msi`,
  `sentinel-2b_msi`, or `sentinel-2c_msi` as a source sensor
- do not reuse `schema_version: 3.0.0` runtimes with `0.6.3`
- install `rsrf>=0.3.1` when you rely on built-in canonical sensors
